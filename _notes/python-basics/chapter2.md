# Variables and Data Types

## Basic Data Types

Python has several basic data types:

1. Integers (int)
2. Floating-point numbers (float)
3. Strings (str)
4. Booleans (bool)

## Examples

```python
# Integer
age = 25

# Float
height = 1.75

# String
name = "Alice"

# Boolean
is_student = True
```
