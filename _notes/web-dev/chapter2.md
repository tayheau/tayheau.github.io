# HTML Fundamentals

## Structure of an HTML Document

Every HTML document follows a basic structure:

```html



    My First Webpage


    Welcome!
    This is my first webpage.


```

## Common HTML Elements

- Headings (`<h1>` to `<h6>`)
- Paragraphs (`<p>`)
- Lists (`<ul>`, `<ol>`, `<li>`)
- Links (`<a>`)
